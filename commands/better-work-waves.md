description: "Better Work waves mode. Explicitly activates wave-based-delivery for project-scale multi-wave work."
argument-hint: "[task description]"
---

Load the `better-work` skill and explicitly activate `wave-based-delivery` for the current task.

Prioritize:

- deciding whether the task is project-scale
- selecting or refreshing the current wave
- keeping project state in `TASK.md`, `PLAN.md`, `MAP.md`, `WAVE.md`, `STATE.md`, and `wave-state.json`
- deciding whether the current wave also needs `round-based-execution`
- preventing project-scale work from collapsing into one linear stream
