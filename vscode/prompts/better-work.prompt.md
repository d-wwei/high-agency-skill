# Better Work

Apply the Better Work protocol to the current task.

## Modes

- `better-work`
- `better-work waves`
- `better-work rounds`
- `better-work verify`
- `better-work unstick`
- `better-work handoff`
- `better-work review`
- `better-work plan`
- `better-work execute`
- `better-work wave-plan`
- `better-work wave-map`
- `better-work wave-status`
- `better-work wave-replan`
- `better-work wave-handoff`

## Rules

1. Investigate before asking the user.
2. Verify before claiming completion.
3. Switch approaches after repeated failure.
4. Use wave-based delivery for project-scale work.
5. Use round-based execution for locally complex work.
6. End with verified completion or a structured handoff.
